# 日本語

```text
{主役の動作末端}と{相手または目標}の間に動作軌道が読める負空間を残す。顔、腕、武器、手足の輪郭を互いに重ねすぎず、背景線は{軌道方向}と競合させない。
```

## 入力変数

`{主役の動作末端}` 刃・拳・足・用具 / `{相手または目標}` 敵・ボール・空間 / `{軌道方向}` 斜め上・水平・円弧

## 使わない条件

密着や組み技では負空間を小さくし、接触点だけを分離する。
