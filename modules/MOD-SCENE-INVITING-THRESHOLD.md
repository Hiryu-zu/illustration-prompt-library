# 役割

未知、行き先、最初の一歩、視点共有を一つの構図へまとめる。

## 日本語

```text
{subject}の{view}から、{threshold}とその先の{destination_hint}を見る構図。道の先は{occlusion}で一部だけ隠すが、手前には到達可能な最初の一歩{first_step}と、使えそうな{tool}を明瞭に置く。人物と観客の視線を同じ行き先へ揃える。
```

## 英語

```text
From a {view} of {subject}, frame {threshold} and the distant hint of {destination_hint}. Partially conceal the route with {occlusion}, while clearly showing the attainable first step, {first_step}, and the usable {tool} in the foreground; align the character's and viewer's gaze toward the same destination.
```

## 入力変数

| 変数 | 意味 | 例 |
|---|---|---|
| `{view}` | 視点 | 肩越し、背中越し |
| `{threshold}` | 境界 | 半開きの巨大門 |
| `{destination_hint}` | 目的地の予告 | 雲上で光る塔 |
| `{occlusion}` | 遮蔽 | 霧と曲がり道 |
| `{first_step}` | 最初の到達点 | 手前の浮遊石 |
| `{tool}` | 使える道具 | 光り始めた地図 |

## 競合・併用注意

目的地を完全に見せず、同時に経路を完全に隠さない。
