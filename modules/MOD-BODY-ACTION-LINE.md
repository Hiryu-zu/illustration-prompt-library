# 役割

全身の動作方向を一本に統合する。

## 日本語

```text
頭から胸郭、骨盤、{endpoint}までを一本の{curve}の主曲線で束ね、全身が{direction}へ動く明確なシルエット。細部の流れは主曲線より弱くする。
```

## 英語

```text
A clear {curve}-shaped line of action connecting the head, ribcage, pelvis, and {endpoint}, with the entire silhouette moving toward {direction}; secondary contours remain subordinate.
```

## 入力変数

| 変数 | 意味 | 例 |
|---|---|---|
| `{endpoint}` | 主曲線の終点 | 支持脚、拳、刃先 |
| `{curve}` | 曲線 | C字、S字 |
| `{direction}` | 動作方向 | 画面右上 |

## 競合・併用注意

互いに逆方向の強い背景線を置かない。
