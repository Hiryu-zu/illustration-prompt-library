# 役割

最初に主役、次に意味のある手掛かりを読ませる。

## 日本語

```text
一読目の焦点は{primary}。最も明るい点と唯一の{accent_color}をそこへ集め、背景は低彩度・低コントラストにする。{gaze_lines}を二読目の焦点{secondary}へ向けるが、二次焦点は小さく控えめに保つ。
```

## 英語

```text
The immediate focal point is {primary}, holding the brightest value and the only {accent_color} accent against a subdued low-contrast background. Direct {gaze_lines} toward the smaller secondary focal point, {secondary}.
```

## 入力変数

| 変数 | 意味 | 例 |
|---|---|---|
| `{primary}` | 主焦点 | 主人公の顔と手 |
| `{secondary}` | 二次焦点 | 半開きの扉 |
| `{gaze_lines}` | 誘導要素 | 目線、腕、床板の線 |
| `{accent_color}` | 差し色 | 琥珀色 |
