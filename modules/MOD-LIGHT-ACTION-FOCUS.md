# 日本語

```text
背景を低彩度・低コントラストに落とし、{focus}だけに短い{accent_color}の発光を置く。逆光で人物の外周を分離し、顔、手、主武器の形は白飛びさせない。
```

## 入力変数

`{focus}` 命中点・技の起点・発火点 / `{accent_color}` 背景への補色
