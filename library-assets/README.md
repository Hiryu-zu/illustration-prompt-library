# アセット管理

このフォルダは、書庫プロンプトと組み合わせる素材を保存する場所。

最初は手動管理でよい。

将来的には、ビューワーから選択して最終生成プロンプトへ統合する。

## アセット種別

- `character`: 専用キャラ
- `costume`: 衣装
- `background`: 背景・舞台
- `world`: 世界観
- `lighting`: ライティング
- `composition`: 構図
- `negative`: NG指定
- `reference_image`: 参考画像
- `finished_image`: 完成イラスト

## 統合方針

アセットは、書庫プロンプト本文に直接混ぜ込まない。

生成時にユーザーがオンにしたものだけを最終プロンプトへ追加する。
